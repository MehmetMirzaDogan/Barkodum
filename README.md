# Barkodum

Türkiye'deki marketler için barkod yönetim uygulaması.

## Özellikler

- 4 ana market desteği (Migros, BİM, ŞOK, A101)
- Özel market ekleme
- Kategori yönetimi
- Barkod arama ve listeleme
- Dark/Light tema
- Fabrika ayarları

## Kurulum

```bash
flutter pub get
flutter run
```
